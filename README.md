# 原神网页插件 (AstrBot Plugin)

一个自动打开原神网页的 AstrBot 插件，支持插件载入时自动打开原神官网，也可通过指令手动打开。

## 📦 功能特点

- ✨ **自动打开**：插件载入时自动打开原神官网
- 🎯 **手动触发**：通过 `/yuanshen` 指令手动打开原神网页
- 📱 **简单易用**：无需复杂配置，安装即可使用
- 🔧 **轻量高效**：代码简洁，运行流畅

## 🚀 安装方法

1. 下载插件文件到 AstrBot 的插件目录
2. 重启 AstrBot 即可自动加载插件

## 📖 使用说明

### 自动打开
插件载入时会自动打开原神官网：`https://ys.mihoyo.com/`

### 手动打开
在聊天窗口发送指令：
```
/yuanshen
```
插件会打开原神官网并回复确认消息。
